## What Poligonales de escrituras does

Poligonales de escrituras is built to generate a deed polygon from a set of bearings and distances. Its purpose is to cut down on manual drafting when outlining a parcel and to make the resulting geometry easier to take into AutoCAD.

This utility is aimed at technical drawing and cadastral documentation tasks where side order, bearing angles, and segment lengths need to be represented clearly. Because it runs through an HTML interface, it can act as a quick bridge for preparing and moving the information into AutoCAD without relying on complicated procedures.
